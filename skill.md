## 二、预测方法论（必须遵守）

按以下权重综合评估，并在 keyFactors 中体现主要依据：

1. **近期状态（40%）**：预选赛表现、大赛成绩、核心球员状态与伤病
2. **球队硬实力（30%）**：阵容厚度、世界排名档位、历史大赛底蕴
3. **历史交锋（15%）**：两队过往交手记录（可用你的常识，不确定要说明）
4. **情境因素（15%）**：东道主优势（美/加/墨三队主场作战）、气候/场地、淘汰赛抗压经验、阵容年龄结构

规则：
- 三个概率（胜/平/负）为整数，总和必须 = 100
- 小组赛允许平局概率较高；**淘汰赛阶段**平局概率表示"90分钟战平进入加时/点球"，并在分析中说明谁更可能笑到最后
- 实力悬殊也不要给出超过 85% 的胜率（足球有偶然性）
- 资料里没有的球队信息，按同档球队估算并注明"估算"
- 本文档"近期状态"数据截至 2026 年 6 月初

## 三、输出格式（严格 JSON，禁止输出任何其他文字）

```json
{
  "teamA": { "name": "队名", "winProb": 45 },
  "draw": 25,
  "teamB": { "name": "队名", "winProb": 30 },
  "predictedScore": "2-1",
  "confidence": "高",
  "keyFactors": ["要点1", "要点2", "要点3"],
  "analysis": "150字以内的综合分析，口吻像专业解说，给出明确倾向。",
  "playersToWatch": [
    { "team": "队名", "player": "球员名", "reason": "一句话理由" }
  ]
}
```

- confidence 取值：高 / 中 / 低
- keyFactors 3~5 条，每条 15 字以内
- playersToWatch 每队各 1 人，共 2 人
