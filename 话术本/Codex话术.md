# Codex 话术本

## 基本用法

在项目目录下直接运行 `codex`，用大白话说你想干什么。

## 话术公式

```
帮我在 [哪个文件/目录] 里 [干什么]，[具体要求]，[不改什么]
```

## Remotion 项目常用话术

### 改模板
```
帮我把 prompt-to-video 的配音从 ElevenLabs 换成 Edge TTS
只改 cli/ 目录下的逻辑，其他不动
```

### 加功能
```
帮我在 src/skills/ 目录下加一个新技能模块叫 "countdown"
专门处理倒计时动画，模板风格参考已有的 charts 模块
```

### 调整输出
```
帮我把视频输出比例从竖屏 1080x1920 改成横屏 1920x1080
所有场景都要改，不只是配置文件
```

### 改视觉风格
```
帮我把代码生成默认的配色方案改成黑金风格
背景深黑色，文字金色，强调元素用暗金色
```

## Claude Code 话术

### 读协作盒子
```
读协作盒子收件箱
```
→ 我会读取 `收件箱/` 下所有新文件，告诉你主公想干什么

### 执行协作盒子里的方案
```
按进行中/AI宣传片方案.md 执行
```
→ 我会读取方案，开始干活

### 记录经验
```
存档 → 技能配方
```
→ 把这次学到的经验写到 `技能配方/` 里

## 双 AI 协作话术

当你想让 Claude Code 和 Codex 一起干活时：

```
"Claude 你读方案定架构，Codex 你按架构写代码"
"Codex 你先生成一版，Claude 你审查给修改意见"
```
